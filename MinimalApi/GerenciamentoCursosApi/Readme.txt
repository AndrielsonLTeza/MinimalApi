Passo 1: Instale o .NET SDK (versão 7.0 ou superior) se ainda não tiver

Baixe em: https://dotnet.microsoft.com/download


Instale as extensões do VS Code:

C# Dev Kit (Microsoft)
C# (Microsoft)
.NET Core Tools

Passo 2: Instale package
dotnet add package Swashbuckle.AspNetCore


Passo 3: Executando o Projeto

Execute o projeto com o comando:
bashdotnet run

Acesse o Swagger para testar a API:

Abra o navegador e acesse: https://localhost:7xxx/swagger
